WHAT DOES THIS DO?
--------------------------------------------------------------------------------
This module overrides the time field on all date_popup fields with a timepicker widget.


REQUIREMENTS
--------------------------------------------------------------------------------
Libraries and the jquery datetimepicker library from Trent Richardson.
Please see Installation for more details.



INSTALLATION
--------------------------------------------------------------------------------
1. Download
https://github.com/trentrichardson/jQuery-Timepicker-Addon
Put them in the libraries/all/jquery-ui-timepicker/ folder

Current supported version is 1.4

Final output should be:
sites/all/libraries/jquery-ui-timepicker/jquery-ui-timepicker-addon.js

2. Configure
Set any field to date_popup with a time granularity of atleast hours and you will see a timepicker when you click into the time field instead of the default text only javascript supported option.
